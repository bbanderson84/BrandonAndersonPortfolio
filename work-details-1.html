About Project

This application consists of a 10 question music quiz. The user has 70 seconds to answer as many questions as possible, once the time runs up the score is calculated and the user is shown how many they got right. If the user does not complete the quiz on time, unanswered questions are marked incorrect.

The language used to create this application can be seen in the list below.

  • Bootstrap CSS.
  • JQuery.
  • Javascript.
